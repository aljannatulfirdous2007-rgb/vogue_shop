7. **Check Shipping** → Switch to SHIPPING tab

## 📊 **TECHNICAL IMPLEMENTATION**

### **React Components:**
- `ProductDetailPageV2.js` - Main component
- State management with hooks
- Axios for API calls
- React Router for navigation

### **API Integration:**
- `GET /api/products/:id/` - Fetch product details
- `GET /api/products/:id/reviews/` - Fetch reviews
- `POST /api/products/:id/reviews/` - Submit review
- `POST /api/cart/:id/add-item/` - Add to cart

### **Features Coded:**
- Tab switching (no page reload)
- Dynamic image gallery
- Size/color selection state
- Quantity selector
- Review form with validation
- Authentication checks
- Loading states
- Error handling
